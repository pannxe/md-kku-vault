# Pharmacokinetic
- **~90% remains in lumen** -> Tx. luminal infection
- Long T1/2 ~11-14 hr.
- Metabolized by glucoronidation
- Excreted in faeces

# Clinical Use
- [[Entamoeba histolytica]] (luminal infection)
- [[Dientamoeba fragilis]]

# ADR
- **[[Diarrhoea]]** -- usually self-limited
- **[[Iodine Toxicity]]**
- [[Nausea and Vomitting]]
- [[Anorexia]]
- [[Headache]]
- Skin rash
