# Causes
- **[[Constipation]]**
- **[[Pregnancy]]**
- [[Portal Hypertension]]
- Straining from [[Defaecation]]

# Pathogenesis
 **Incr. venous pressure in rectal plexus**
- At [[Superior Rectal Plexus]] -> **internal haemorrhoid**
- At [[Inferior Rectal Plexus]] -> **extenal haemorrhoid**
- Can be both
