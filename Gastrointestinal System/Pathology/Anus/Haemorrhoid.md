# Causes
- **[[Constipation]]**
- **[[Pregnancy]]**
- [[Portal Hypertension]]
- Straining from [[Defaecation]]

# Pathogenesis
 **↑ [[Venous Pressure]] in rectal plexus**
- At [[Superior Rectal Plexus]] -> **internal haemorrhoid**
- At [[Inferior Rectal Plexus]] -> **extenal haemorrhoid**
- Can be both

# Consequences
- Painful ulceration → **Haemorrhage**
- **[[Thrombosis]]**
