**Vasoactive intestinal peptide**

# Source 
**Entire GI**, **[[Pancreas]]**, and **[[Brain]]**

# Function
- **[[Stomach]]** -> **inh. [[Gastric Secretion]]** secretion
- **[[Small Intestine]]** -> **inh. absorption**
- **[[Brain]]** -> **[[Circadian Rhythm]]** 
- **[[Smooth Muscle]]** -> **relaxation**
> e.g., blood vessel
