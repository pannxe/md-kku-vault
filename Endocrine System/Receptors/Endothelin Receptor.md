# Type B
- At both [[Vascular Smooth Muscle Cell]] and [[Endothelial Cells]]
- Equal potency for all types of [[Endothelins]]

- **落 [[NO Synthase#3]]** → **↑ [[NO]]**

→ **[[Vasoconstriction]]**, ↓ [[Apoptosis]], etc.
