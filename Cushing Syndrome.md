# Pathogenesis
**Excess [[Glucocorticoids]]** ←
- Exogenous corticosteroid ← drug
- Endogenous ←
	- **↑ [[ACTH]]** ← **[[Cushing Disease]]**, [[Small-cell Carcinoma of Lung]], [[Thymoma]], etc.
	- **↑ [[Cortisol]]** ← **[[Adrenal Cortex Adenoma]]**
